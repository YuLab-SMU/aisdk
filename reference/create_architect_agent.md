# Create Architect agent for team composition

Create Architect agent for team composition

## Usage

``` r
create_architect_agent(agent_library, model = "claude-3-5-sonnet-20241022")
```

## Arguments

- agent_library:

  AgentLibrary instance with discovered agents

- model:

  Model to use for Architect (default: sonnet)

## Value

Agent object configured as Architect
