# Create Skill Scaffold

Create a new skill project with the standard structure.

## Usage

``` r
create_skill(name, path = ".", author = NULL, description = NULL)
```

## Arguments

- name:

  Skill name.

- path:

  Directory to create the skill in.

- author:

  Author name.

- description:

  Brief description.

## Value

Path to the created skill directory.
