# Create Architect V2 agent

Create Architect V2 agent

## Usage

``` r
create_architect_v2(agent_library, model = "claude-3-5-sonnet-20241022")
```

## Arguments

- agent_library:

  AgentLibrary instance with discovered agents

- model:

  Model to use for Architect (default: sonnet)

## Value

ArchitectV2 R6 object
