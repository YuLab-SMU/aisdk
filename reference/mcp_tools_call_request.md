# Create MCP tools/call request

Create MCP tools/call request

## Usage

``` r
mcp_tools_call_request(
  tool_name,
  arguments = structure(list(), names = character(0)),
  id
)
```

## Arguments

- tool_name:

  The tool name

- arguments:

  Tool arguments as a list

- id:

  Request ID

## Value

A JSON-RPC request
